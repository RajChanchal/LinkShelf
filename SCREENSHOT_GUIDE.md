# App Store Screenshot Guide for LinkShelf

## Required Screenshot Sizes
You need screenshots in one of these resolutions:
- **1280 x 800px** (recommended for smaller displays)
- **1440 x 900px** (recommended)
- **2560 x 1600px** (for Retina displays)
- **2880 x 1800px** (for high-resolution displays)

## Recommended Screenshots (3-10 total)

### 1. Main View - Link List (Essential)
- Show the popover open with a list of links
- Display 3-5 sample links
- Show the header with "LinkShelf" title and "+" button
- **What to show**: Clean interface with organized links

### 2. Adding a Link (Recommended)
- Show the "Add Link" sheet/modal open
- Display the form with Title and URL fields
- **What to show**: Easy link management interface

### 3. Copying a Link (Recommended)
- Show a link with the checkmark (copied state)
- **What to show**: Visual feedback when copying

### 4. Empty State (Optional)
- Show the empty state message when no links exist
- **What to show**: First-time user experience

### 5. Menu Bar Context (Optional)
- Show the menu bar with the LinkShelf icon
- Show the popover open from the menu bar
- **What to show**: How the app appears in the menu bar

## How to Take Screenshots

### Method 1: Using macOS Screenshot Tool
1. Open LinkShelf and position the popover where you want it
2. Press `Shift + Command + 4` to enter screenshot mode
3. Press `Spacebar` to switch to window capture mode
4. Click on the popover window to capture it
5. The screenshot will be saved to your Desktop

### Method 2: Using the Screenshot Script
Run the provided script to help capture screenshots at specific resolutions.

### Method 3: Using Xcode Simulator (if testing)
1. Run the app in Xcode
2. Use the simulator's screenshot feature
3. Adjust window size to match required resolutions

## Tips for Great Screenshots

1. **Use Real Data**: Add 3-5 realistic links (LinkedIn, GitHub, Portfolio, etc.)
2. **Clean Interface**: Make sure the popover looks polished
3. **Show Key Features**: Highlight copy, open, and add functionality
4. **Consistent Style**: Use the same background/context for all screenshots
5. **High Quality**: Ensure text is readable and UI elements are clear

## Editing Screenshots

After capturing, you may need to:
- Resize to exact dimensions (1280x800, 1440x900, etc.)
- Add subtle annotations or highlights (optional)
- Ensure consistent styling across all screenshots

You can use:
- **Preview** (built-in macOS app) - for resizing
- **Photoshop/GIMP** - for advanced editing
- **Sketch/Figma** - for design consistency
